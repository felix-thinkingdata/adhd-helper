# Focus — ADHD Helper 使用手册

## 简介

**Focus** 是一款专为 ADHD（注意力缺陷多动障碍）设计的 macOS 菜单栏工具。它常驻在屏幕顶部的菜单栏中，提供四个核心功能帮助你对抗注意力不集中的问题。

---

## 安装

### 方式一：从源码构建

**前置条件**：
- macOS 12+ (Apple Silicon 或 Intel)
- Node.js 18+
- Rust 工具链 (`rustup`)

```bash
# 1. 克隆项目
git clone <repo-url>
cd adhd-helper

# 2. 安装前端依赖
npm install

# 3. 开发模式运行
npm run tauri dev

# 4. 或者构建生产版本
npm run tauri build
```

构建完成后，应用位于：
```
src-tauri/target/release/bundle/macos/Focus.app
```

### 首次打开

由于应用未签名，macOS 可能阻止打开。处理方式：

1. 双击 `Focus.app`
2. 如提示"无法验证开发者"，打开 **系统设置 → 隐私与安全性**
3. 点击 **"仍然允许"**
4. 或者在终端运行：`xattr -cr /path/to/Focus.app`

## 基本使用

### 启动应用

打开 Focus 后，应用图标会出现在 **屏幕右上角菜单栏**（时钟旁边）。Dock 栏不会显示图标。

### 打开操作面板

**点击菜单栏图标** → 弹出操作面板

操作面板是一个 320x480 的小窗口，始终显示在所有窗口最前面。

### 关闭操作面板

- **点击面板外的任何区域** → 自动收起（推荐）
- 或者再次 **点击菜单栏图标**

> 关闭面板不影响计时器运行。计时器在后台持续工作。

### 右键快捷菜单

**右键点击菜单栏图标** 可快速操作：

| 菜单项 | 说明 |
|--------|------|
| 开始专注 25min | 不用打开面板直接开始专注 |
| 暂停 / 继续 / 停止 | 计时器控制（根据状态自动灰显） |
| 5/10/30 分钟后提醒 | 快速添加提醒 |
| 退出 | 关闭应用 |

## 四大功能

### 1. 番茄钟 (Timer)

#### 使用方法

1. 点击底部 **⏱ Timer** 标签
2. 点击 **Start Focus** 开始专注计时
3. 倒计时在托盘图标标题中实时显示（如 `24:59`）
4. 专注结束后自动切换到休息倒计时
5. 休息结束后回到空闲状态

#### 操作按钮

| 状态 | 可用按钮 | 说明 |
|------|---------|------|
| 空闲 | Start Focus | 开始 25 分钟专注 |
| 空闲 | Start Break | 手动开始 5 分钟休息 |
| 运行中 | Pause | 暂停计时 |
| 暂停中 | Resume | 恢复计时 |
| 运行/暂停 | Stop | 停止并重置 |

#### 界面说明

- **圆环进度条**：红色 = 专注中，绿色 = 休息中，灰色 = 空闲
- **中间数字**：剩余时间 `MM:SS`
- **下方文字**：FOCUS / BREAK / READY
- **托盘图标**：实时显示 `MM:SS`

#### 专注检查

专注期间，应用每隔 N 分钟会弹出 macOS 系统通知：

> **Focus Check-in**
> Are you still working on "写项目方案"?

帮助你拉回飘走的注意力。

---

### 2. 任务聚焦 (Tasks)

#### 使用方法

1. 点击底部 **☐ Tasks** 标签
2. 在输入框输入任务名称，按 **Enter** 或点击 **Add**
3. 页面只显示 **当前一个任务**
4. 完成后点击 **Done** → 自动切换到下一个
5. 暂时不想做？点击 **Skip** → 跳到下一个

#### 为什么只显示一个？

ADHD 的大脑面对多个任务容易"瘫痪"——不知道从哪个开始。Focus 故意隐藏其他任务，让你只看到当前一个。每次只需要做一个决定：**做还是跳过？**

#### 任务与计时器联动

当你在 Tasks 页面选中任务并启动专注时，计时器页面会内联显示当前任务名。专注检查通知也会包含任务名。

---

### 3. 快速提醒 (Alerts)

#### 使用方法

1. 点击底部 **🔔 Alerts** 标签
2. 输入提醒内容（如"站起来喝水"）
3. 点击快捷按钮选择时间：

| 按钮 | 时间 |
|------|------|
| 5m | 5 分钟后 |
| 10m | 10 分钟后 |
| 30m | 30 分钟后 |
| 1h | 1 小时后 |

4. 到时后收到 macOS 系统通知

#### 使用场景

- "30 分钟后提醒我回消息"
- "10 分钟后提醒我关掉社交媒体"
- "5 分钟后提醒我站起来走走"

也可以通过 **右键托盘图标** 快速添加 5/10/30 分钟提醒（默认文案）。

> 注意：当前版本提醒不跨重启保存。重启应用后未触发的提醒会丢失。

---

### 4. 设置 (Settings)

点击底部 **⚙ Settings** 标签可调整：

| 设置项 | 默认值 | 可选值 |
|--------|--------|--------|
| Focus Duration（专注时长） | 25 分钟 | 15 / 20 / 25 / 30 / 45 分钟 |
| Break Duration（休息时长） | 5 分钟 | 3 / 5 / 10 / 15 分钟 |
| Check-in Interval（检查间隔） | 10 分钟 | 5 / 10 / 15 / 20 分钟 |
| Sound Notifications（声音） | 开启 | 开 / 关 |

设置自动保存，下次打开应用会记住你的选择。

## 推荐工作流

针对 ADHD 的典型工作流程：

### 早上启动

1. 打开 Focus
2. 进入 Tasks 页面，快速添加今天要做的 3-5 个任务
3. 点击当前任务 → 自动切换到第一个

### 专注循环

1. 点击 **Start Focus** 开始 25 分钟
2. 专注期间如果收到检查通知——确认自己在做正确的事
3. 专注结束 → 自动进入 5 分钟休息
4. 休息结束 → 完成当前任务（Done）或跳过（Skip）
5. 点击 **Start Focus** 开始下一轮

### 随时提醒

- 想到要做但怕忘的事 → Alerts 页面一键添加
- 不用切换到日历或提醒事项应用

## 常见问题

### Q: 关闭面板后计时器还在运行吗？

**是的。** 计时器在 Rust 后台线程运行，不受 WebView 影响。托盘图标的时间会持续更新。

### Q: 为什么 Dock 栏看不到图标？

这是故意的设计。Focus 是一个菜单栏工具，只在菜单栏显示图标，不出现在 Dock 中。这样不会占用 Dock 空间，也不会被误关闭。

### Q: 专注结束后自动开始休息吗？

**是的。** 专注 25 分钟结束后，会自动进入 5 分钟休息倒计时。休息结束后回到空闲状态，需要手动开始下一轮专注。

### Q: 任务数据会丢失吗？

**不会。** 任务列表和设置通过本地文件持久化保存（`app-store.json`）。重启应用后数据还在。

### Q: 为什么打开应用时 macOS 提示"无法验证开发者"？

因为应用没有 Apple Developer 签名。打开 系统设置 → 隐私与安全性 → 点击"仍然允许"即可。

### Q: 如何退出应用？

右键点击菜单栏图标 → **退出**。
